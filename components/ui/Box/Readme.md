This box is amazing, use it responsibly.

### Examples

<strong>Theme</strong>

```jsx
<template>
  <div class="container">
    <div class="theme--dark">
      <Box>
        <div class="box-wrapper">
          <Label>Dark theme!</Label>
        </div>
      </Box>
    </div>
    <div class="theme--light">
      <Box>
        <div class="box-wrapper">
          <Label>Light theme!</Label>
        </div>
      </Box>
    </div>
  </div>
</template>

<style>
.container {
  display: flex;
  justify-content: space-evenly;
}
.box-wrapper {
  padding: 50px;
}
</style>
```
