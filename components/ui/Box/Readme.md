This box is amazing, use it responsibly.

### Examples

<strong>Theme</strong>

```jsx
<template>
  <div class="container">
    <div class="theme--dark">
      <Box>
        <Label>Dark theme!</Label>
      </Box>
    </div>
    <div class="theme--light">
      <Box>
        <Label>Light theme!</Label>
      </Box>
    </div>
  </div>
</template>

<style>
.container {
  display: flex;
  justify-content: space-evenly;
}
</style>
```
