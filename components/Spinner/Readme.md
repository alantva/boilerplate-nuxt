### How to use

You can use this spinner as in this example.

```jsx
<template>
  <div class="theme--light">
    <div class="wrapper">
      <Spinner></Spinner>
      <Label>Carregando</Label>
    </div>
  </div>
</template>
<style>
.wrapper {
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
}
</style>
```
